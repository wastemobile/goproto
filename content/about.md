Hugo.

Hugo also offers powerful support for code snippets:

```
package main
import "fmt"
func print_hi(name string) {
  fmt.Println("Hi, ", name)
}

func main() {
  print_hi("Tom")
}
//=> prints 'Hi, Tom' to STDOUT.
```
